# Instagram Addiction & Mental Wellness Analysis

## Project Overview

This project analyzes the impact of Instagram usage on:

- Sleep patterns
- Productivity
- Mental wellness
- Anxiety and self-comparison behavior

The dataset was synthetically generated for educational and portfolio purposes.

---

## Objectives

- Analyze Instagram screen time behavior
- Understand productivity loss trends
- Study sleep impact caused by excessive social media usage
- Explore mental wellness indicators
- Build storytelling visualizations using Python

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset Features

The dataset includes:

- Daily screen time
- Sleep hours
- Productivity loss
- Anxiety levels
- Focus ratings
- Mood after usage
- Reels consumption
- Addiction levels

---

## Key Insights

### 1. High Screen Time Reduces Sleep

Users with higher Instagram usage generally reported fewer sleep hours.

### 2. Productivity Loss Increases with Addiction

Highly addicted users showed greater productivity loss and reduced focus levels.

### 3. Mental Wellness Impact

Anxiety and self-comparison scores were positively correlated with screen time.
